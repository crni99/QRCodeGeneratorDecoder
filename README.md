# QR Code Generator Decoder

- Spring Boot
- Spring MVC
- Thymeleaf
- ZXing barcode scanning library for Java

# Docker
- Pull docker image
```
docker pull crni99/qrcodegeneratordecoder
```
- Run docker and expose port
```
docker run -d -p 8080:8080
```
- Open a web browser and paste this:
```
http://localhost:8080/
```
